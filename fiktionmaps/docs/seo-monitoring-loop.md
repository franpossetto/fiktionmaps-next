# SEO Monitoring Loop

## Weekly Cadence
Run this routine once per week.

## KPIs
- Indexed pages count (public routes only).
- Impressions and clicks by top landing pages.
- Average CTR on `/fictions`, `/map`, `/scenes`.
- Core Web Vitals pass rate for mobile.

## Search Console Review
- Coverage issues by category.
- International targeting consistency (`en` and `es` alternates).
- Manual action/security issue status.

## Technical Regression Checklist
- `robots.txt` still allows public routes.
- `sitemap.xml` includes all public locale routes.
- Public pages still expose canonical + alternates.
- Private pages keep `noindex`.

## Backlog Template
For each issue, capture:
- URL or URL pattern
- Issue type
- Impact (high/medium/low)
- Owner
- Target release
